# 瑶族鼓王游戏 🥁

一款以瑶族文化传承为主题的网页游戏，让玩家在游戏中体验瑶族的传统文化。

## 🎮 游戏简介

"盘王鼓台试炼"是一款充满瑶族文化气息的网页游戏，玩家将穿越瑶山，学习瑶族语言、工艺、艺术，最终成为真正的文化传承者。

## 🌟 游戏特色

- **7个精心设计的关卡**：长鼓森林、瑶语符文、盘王祭坛、瑶寨刺绣、盘王节庆典、银饰工艺、文化传承
- **瑶族文化元素**：喀斯特山峰、吊脚楼、刺绣边框、传统纹样
- **精美视觉效果**：使用瑶族传统色彩（蓝靛色、朱红色、金黄色、翠绿色）
- **完整音效系统**：背景音乐、点击音效、胜利音效、失败音效
- **丰富动画效果**：银饰光泽、长鼓舞动作、刺绣织造等特色动画

## 📖 游戏玩法

### 关卡介绍

1. **第1关：长鼓森林** - 穿越瑶山森林，收集长鼓能量，躲避山间石块
2. **第2关：瑶语符文** - 学习瑶族语言，点击符文解锁文化密码
3. **第3关：盘王祭坛** - 挑战鼓王试炼，用长鼓舞技艺获得认可
4. **第4关：瑶寨刺绣** - 学习瑶族刺绣工艺，传承古老技艺
5. **第5关：盘王节庆典** - 参与盘王节活动，体验瑶族传统节庆
6. **第6关：银饰工艺** - 学习瑶族银饰制作，传承精湛技艺
7. **第7关：文化传承** - 综合运用所学知识，成为真正的文化传承者

### 操作方式

- **键盘**：左右方向键移动角色
- **触摸**：移动端显示控制按钮
- **鼠标**：点击各种互动元素

## 🎨 设计理念

游戏采用了瑶族传统文化元素：
- **背景设计**：喀斯特地貌山峰、云雾效果、吊脚楼轮廓
- **色彩系统**：蓝靛色、朱红色、金黄色、翠绿色等传统色彩
- **纹样装饰**：蝴蝶纹、八角星纹、凤凰纹、几何纹等传统图案

- **动画**：CSS3 Animations + Keyframes
- **响应式设计**：适配桌面端和移动端

## 🌏 瑶族文化

瑶族是中国南方的重要少数民族之一，有着悠久的历史和丰富的文化遗产：
- **长鼓舞**：瑶族最富特色的舞蹈形式，已被列入国家级非物质文化遗产
- **盘王节**：瑶族最重要的传统节日，每年农历十月十六举行
- **瑶族刺绣**：色彩鲜艳、图案精美，被誉为"穿在身上的史诗"
- **瑶族银饰**：工艺精湛，寓意吉祥，是重要的文化符号
